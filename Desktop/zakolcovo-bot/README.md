
# Сервер для Telegram-бота ресторана "Закольцово"

Это серверное приложение (backend), которое управляет вашим Telegram-ботом. Оно написано на Node.js и TypeScript. Следуйте этой инструкции, чтобы запустить его на вашем компьютере.

---

### **⚠️ ВАЖНОЕ ПРЕДУПРЕЖДЕНИЕ О БЕЗОПАСНОСТИ**

Вы опубликовали ваш токен Telegram-бота. Это очень опасно, так как любой может получить контроль над вашим ботом.

**НЕМЕДЛЕННО СДЕЛАЙТЕ СЛЕДУЮЩЕЕ:**
1.  Откройте Telegram и найдите бота **@BotFather**.
2.  Отправьте ему команду `/mybots`.
3.  Выберите вашего бота из списка.
4.  Нажмите на кнопку **"API Token"**.
5.  Нажмите **"Revoke current token"** (отозвать текущий токен). Старый токен перестанет работать.
6.  BotFather сразу же выдаст вам **НОВЫЙ ТОКЕН**.
7.  **Никогда и никому не показывайте ваш новый токен!** Именно его вы будете использовать дальше.

---

### Шаг 1: Подготовка (установка Node.js)

Для запуска сервера вам понадобится **Node.js**.
1.  Перейдите на официальный сайт: [https://nodejs.org/](https://nodejs.org/)
2.  Скачайте и установите **LTS** версию. Это стандартная, стабильная версия.
3.  После установки откройте терминал (командную строку) и проверьте, что все установилось, выполнив команды:
    ```bash
    node -v
    npm -v
    ```
    Вы должны увидеть версии `node` и `npm`.

---

### Шаг 2: Получение ключей доступа (API Keys)

Вам понадобятся два ключа:
1.  **Telegram Bot Token:** Вы получили его на шаге с предупреждением о безопасности от **@BotFather**.
2.  **Google Gemini API Key:** Этот ключ нужен для работы искусственного интеллекта.
    *   Перейдите на сайт [Google AI Studio](https://aistudio.google.com/app/apikey).
    *   Войдите с помощью вашего Google-аккаунта.
    *   Нажмите **"Create API key"**.
    *   Скопируйте сгенерированный ключ.

---

### Шаг 3: Настройка проекта

1.  **Создайте папку** для проекта на вашем компьютере, например, `zakolcovo-bot`.
2.  **Скопируйте все файлы** из предоставленного XML в эту папку.
3.  **Создайте файл `.env`**:
    *   В папке проекта найдите файл `.env.example`.
    *   Создайте его копию и переименуйте ее в `.env`.
4.  **Заполните файл `.env`**:
    *   Откройте файл `.env` в текстовом редакторе.
    *   Вставьте ваши ключи, которые вы получили на Шаге 2. Должно получиться так:
        ```
        # Вставьте ваш НОВЫЙ токен от BotFather
        TELEGRAM_BOT_TOKEN=72...ВАШ_НОВЫЙ_ТОКЕН..._A
        
        # Вставьте ваш ключ от Google Gemini
        API_KEY=AIzaSy...ВАШ_GEMINI_КЛЮЧ...
        ```
5.  **Установите зависимости**:
    *   Откройте терминал (командную строку) в папке вашего проекта.
    *   Выполните команду:
        ```bash
        npm install
        ```
    *   Эта команда скачает все необходимые библиотеки для работы бота (`node-telegram-bot-api`, `dotenv` и др.).

---

### Шаг 4: Запуск бота

1.  Убедитесь, что вы находитесь в папке проекта в вашем терминале.
2.  Выполните команду для запуска сервера:
    ```bash
    npm start
    ```
3.  Если все сделано правильно, вы увидите в терминале сообщение: `[BOT] Zakolcovo Bot запущен...`.
4.  **Готово!** Теперь откройте Telegram, найдите вашего бота и начните с ним общаться.

**Важно:**
*   Окно терминала, где вы запустили команду `npm start`, **должно оставаться открытым**. Как только вы его закроете, ваш бот перестанет работать.
*   Это локальный запуск. Для того чтобы бот работал 24/7, его нужно будет разместить на хостинге (например, Render, Heroku, Vercel), но для начала и тестирования запуск на своем компьютере — идеальный вариант.

### **Устранение неполадок**

**Проблема:** При выполнении `npm install` я получаю ошибку `ETARGET` или `No matching version found for @google/genai`.

**Решение:** Это означает, что версия пакета, указанная в `package.json`, устарела.
1.  Откройте файл `package.json`.
2.  Найдите строку `"@google/genai": "^0.1.0"`.
3.  Замените ее на `"@google/genai": "^0.12.0"`.
4.  Сохраните файл и снова запустите `npm install`.
