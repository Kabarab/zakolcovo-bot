# Руководство пользователя: Профессиональный Антидетект-Браузер

Это приложение предназначено для управления множеством независимых браузерных профилей с уникальными цифровыми отпечатками. Оно идеально подходит для арбитража трафика, фарма аккаунтов и обеспечения анонимности.

---

## 🚀 1. Быстрый старт

1.  **Установка зависимостей**:
    ```bash
    npm install
    npx playwright install chromium
    ```
2.  **Запуск приложения**:
    ```bash
    npm run dev
    ```

---

## 📂 2. Управление профилями

### Создание одиночного профиля

1. Нажмите кнопку **"Создать"** (+).
2. Введите имя профиля.
3. В разделе **"Прокси"** укажите данные (Host, Port, User, Pass). Нажмите **"Синхронизировать по IP"**, чтобы автоматически настроить таймзону и геолокацию.
4. В разделе **"Цифровой отпечаток"** нажмите **"Рандомизировать"**, чтобы получить качественный фингерпринт на базе реальной статистики (BrowserForge).
5. (Опционально) Импортируйте куки в формате JSON для быстрого входа в аккаунты.

### Массовое создание (Bulk Create)

1. Нажмите **"Массовое создание"** на главном дашборде.
2. Введите базовое имя (например, "Google_Acc") и количество (например, 50).
3. Система автоматически создаст 50 профилей с уникальными именами и случайно сгенерированными, статистически достоверными отпечатками.

---

## 🛡️ 3. Продвинутые настройки стелса

- **WebRTC**:
  - _Подмена (Spoof)_: Рекомендуемый режим. Скрывает ваш реальный IP, заменяя его на внутренний (192.168.1.100).
  - _Отключить_: Полностью блокирует WebRTC (некоторые сайты могут это заметить).
- **Hardware**: Вы можете вручную выбрать количество ядер CPU и объем RAM, которые будет видеть сайт.
- **Платформа**: Переключайтесь между Desktop и Mobile. Мобильные профили активируют эмуляцию тач-скрина и соответствующие User-Agent.

---

## 🤖 4. Автоматизация и Фарм (Warming)

### Использование Ghost Cursor

При запуске профиля автоматически активируется **Ghost Cursor**. Ваши скрипты автоматизации или ручные действия будут дополнены человекоподобными движениями мыши по ломаным кривым, что обходит детекторы ботов.

### Движок прогрева (Warming Engine)

_Функционал в процессе расширения через API:_
Вы можете запланировать задачи для профилей:

- Просмотр YouTube (нагул кук).
- Чтение новостей.
- Поиск в Google.

---

## 💡 Советы и рекомендации

1.  **Качество прокси**: Используйте резидентские прокси (Residential) США или Европы для максимального траста.
2.  **Проверка**: После запуска профиля проверьте его на [Pixelscan.net](https://pixelscan.net) или [CreepJS](https://abrahamjuliot.github.io/creepjs/). Вы должны увидеть зеленые показатели.
3.  **База данных**: Все ваши профили хранятся в надежной базе SQLite (`profiles.db` в папке userData).

---

## 🛠️ Устранение неполадок

- **Браузер не запускается**: Убедитесь, что вы выполнили `npx playwright install chromium`.
- **Прокси не работает**: Проверьте формат (обычно SOCKS5 или HTTP) и корректность логина/пароля.
- **Высокое потребление ресурсов**: При запуске более 10–20 профилей одновременно убедитесь, что у вас достаточно RAM.
